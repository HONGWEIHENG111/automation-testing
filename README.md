🤖 Automated AI Agent Testing & Evaluation Framework
Author: Henry HONG
E-mail:weihehong3-c@my.cityu.edu.hk

A powerful, fully automated framework designed to test, monitor, and evaluate web-based AI Agent platforms. By combining Selenium for UI automation and DeepSeek API for LLM-as-a-Judge evaluation, this tool streamlines the QA process for AI chatbots and document-processing agents.

🌟 Key Features
Automated UI Interaction: Logs in, uploads various document types (PDF, DOCX, TXT, CSV), inputs prompts, and configures Agent modes automatically.

Intelligent Generation Monitoring: Dynamically monitors the DOM to detect when the AI finishes generating text, preventing timeout errors or incomplete data extraction.

LLM-as-a-Judge (DeepSeek Integration): Automatically evaluates the generated answers against the source documents for accuracy, language consistency, and hallucination detection.

Robust Error Handling: Includes an emergency stop hotkey (ESC), automatic popup closing ("mine-sweeping"), and browser-crash recovery.

Comprehensive Reporting: Outputs a detailed evaluation_results.xlsx and utilizes an LLM to dynamically cluster failure reasons into a highly readable Summary.csv.

Multi-State Management: Supports switching between pure text mode, Agent Finder Mode, General Agent Mode, and Agent Master Mode on the fly.

📂 Project Structure
main.py: The core orchestrator. Handles browser initialization, login, file uploading, DOM monitoring, and data extraction.

state_manager.py: Manages the specific UI clicking logic for different platform configurations (Auto Mode, Agent Master searching, etc.).

llm_evaluator.py: Interfaces with the DeepSeek API. Compares the system's generated answer with the uploaded document to evaluate quality, language status, and citation formatting.

summary_generator.py: Post-processing script. Reads the Excel results and asks the LLM to cluster common problems, ultimately generating a clean Summary.csv report.

Testcase Excel Setup:
Create or modify your input Excel file (e.g., Testcase_20260520_1.xlsx). Ensure it has the following headers:

Selected Agent: The specific agent to route the request to (leave blank for pure text/default mode).

🛑 Emergency Stop (Kill Switch)
If the web UI changes unexpectedly or you need to halt the script immediately, press the ESC key. The script will safely interrupt the current task, save any progress it has made to the Excel file, and release browser control.

📊 Outputs
Upon completion, the framework generates two artifacts in the root directory:

evaluation_results.xlsx: A row-by-row breakdown of every test case, including raw answers, generation times, extracted links, and DeepSeek's detailed evaluation.

Summary.csv: An aggregated report generated by DeepSeek, featuring:

Overall Pass/Fail rates.

Failure reason clustering with test IDs.

Language consistency statistics.

Suggested priority fixes.
